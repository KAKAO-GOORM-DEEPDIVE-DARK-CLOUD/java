# 아이템 57. 지역변수의 범위를 최소화화라.

## 1. 왜 지역변수의 범위를 최소화해야 할까?
### - 먼 부분에서 미리 지역변수를 선언부터 해두면 가독성이 떨어지고, 정작 사용하는 시점엔 초깃값이 기억나지 않을 수도 있기 때문이다.

## 2. 그럼 어떻게 지역변수를 사용해야할까?

### - 거의 모든 지역변수는 선언과 동시에 초기화해야한다.
